# CI3-ELOQUENT

A simple eloquent for easier your life.

## HOW TO USE

- Step 1: Run this command in your codeigniter project

```
composer require maika-kanaka/ci3-eloquent
```

- Step 2: Extend your model

```
class User_model extends Eloquent
```

### METHOD

- table

- insert 

- delete 

- update

- primaryKeyInc
